## 🧠 O que aprendi até agora?

### 📈 Análise e Exploração de Dados (EDA)
* **Limpeza de Dados:** Tratamento de valores nulos e duplicados com Pandas.
* **Estatística Descritiva:** Média, mediana, desvio padrão e correlação.
* **Visualização:** Criação de gráficos de dispersão, histogramas e boxplots para entender a distribuição dos dados.

### 🤖 Machine Learning & Modelagem
* **Fundamentos de Redes Neurais:** Implementação e lógica do **Perceptron**.
* **Ciclo de Modelagem:** Divisão de treino/teste, normalização de dados e métricas de avaliação.
* **Prevenção de Problemas:** Identificação e tratamento de **Overfitting** e **Underfitting**.

### 🛠️ Engenharia de Software para Data Science
* Versionamento de experimentos com **Git**.
* Organização de notebooks e ambientes virtuais.
* Manipulação eficiente de arrays com **NumPy**.
